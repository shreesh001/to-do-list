# To-Do List App

A simple and clean to-do list application built using HTML, CSS, and JavaScript.  
It allows users to add, mark, and delete tasks with a minimal and responsive UI.

---

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive design

---

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
